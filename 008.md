## Exercice 1 — Créer l'utilisateur `guest` manuellement

Créez l'utilisateur `guest` en modifiant directement les fichiers système (`/etc/passwd`, `/etc/shadow`, `/etc/group`).

**Étapes et commandes :**
```bash
# 1. Ajouter une entrée dans /etc/passwd
sudo nano /etc/passwd
# Ajouter à la fin :
# guest:x:1100:1100:Guest User:/home/guest:/bin/bash

# 2. Ajouter une entrée dans /etc/shadow
sudo nano /etc/shadow
# Ajouter à la fin (mot de passe désactivé) :
# guest:!:19000:0:99999:7:::

# 3. Ajouter un groupe dans /etc/group
sudo nano /etc/group
# Ajouter à la fin :
# guest:x:1100:

# 4. Créer le répertoire home et lui attribuer le bon propriétaire
sudo mkdir /home/guest
sudo chown guest:guest /home/guest
sudo chmod 755 /home/guest
```

**Vérification :**
```bash
id guest
ls -ld /home/guest
```

## Exercice 5 — Fichiers et groupes

Rétablissez la situation : lucien appartient aux groupes `users`, `comptabilite`. Connectez-vous en lucien.

**Commande(s) :**
```bash
sudo usermod -g users -G comptabilite lucien
su lucien
```

---

Créez un fichier `testgroupe` dans le répertoire personnel de lucien.

**Commande :**
```bash

```

---

Vérifiez les propriétés du fichier `testgroupe`.

**Commande :**
```bash
ls -l ~/testgroupe
```

**Résultat :**
```
-rw-r--r-- 1 lucien users 0 ... testgroupe
```

---

Changez l'appartenance de `testgroupe` au groupe `comptabilite`. Constatations ?

**Commande :**
```bash
chgrp comptabilite ~/testgroupe
ls -l ~/testgroupe
```

**Constatations :**
> Ça fonctionne car lucien appartient au groupe `comptabilite`. Un utilisateur ne peut changer le groupe d'un fichier qu'il possède que vers un groupe dont il est membre.

**Commande :**
```bash
chgrp eleves ~/testgroupe
```

**Constatations :**
> Refus — lucien n'appartient pas au groupe `eleves`, il ne peut donc pas attribuer ce groupe à son fichier.

## Exercice 6 — Partage entre Jack et Joe

Créez les utilisateurs `jack` et `joe` et l'arborescence suivante :

```
/home
├── jack
│   └── projet
│       └── main_jack.cpp
└── joe
    └── projet
        └── main_joe.cpp
```

**Commande(s) :**
```bash
sudo adduser jack
sudo adduser joe
sudo mkdir -p /home/jack/projet
sudo mkdir -p /home/joe/projet
sudo touch /home/jack/projet/main_jack.cpp
sudo touch /home/joe/projet/main_joe.cpp
```

---

Joe et Jack veulent partager leurs répertoires `projet` **uniquement entre eux** (aucun autre utilisateur n'y a accès).

### Demandes à faire à l'administrateur

**Description :**
> Créer un groupe commun (ex: `jackjoe`) et y ajouter jack et joe. Puis changer le groupe propriétaire des répertoires et fichiers concernés.

**Commandes administrateur :**
```bash
sudo groupadd jackjoe
sudo usermod -aG jackjoe jack
sudo usermod -aG jackjoe joe

# Changer le groupe des répertoires projet
sudo chgrp jackjoe /home/jack/projet
sudo chgrp jackjoe /home/joe/projet
sudo chgrp jackjoe /home/jack/projet/main_jack.cpp
sudo chgrp jackjoe /home/joe/projet/main_joe.cpp
```

---

### Commandes que doit taper Jack

```bash
# Appliquer les droits : proprio=rwx, groupe=rwx, autres=---
chmod 770 ~/projet
chmod 660 ~/projet/main_jack.cpp
```

---

### Commandes que doit taper Joe

```bash
chmod 770 ~/projet
chmod 660 ~/projet/main_joe.cpp
```

## Exercice 7 — Script de création/suppression de 100 comptes

Écrivez un script permettant de **créer ou supprimer 100 comptes utilisateurs**.

**Script :**
```bash
#!/bin/bash

# Récupère le premier argument passé au script (create ou delete)
ACTION=$1

# Compteur qui démarre à 1
i=1

# Boucle tant que i est inférieur ou égal à 100
while [ $i -le 100 ]; do

    # Construit le nom d'utilisateur : user1, user2, ... user100
    USERNAME="user$i"

    if [ "$ACTION" = "create" ]; then
        # Crée l'utilisateur avec son répertoire home (-m)
        useradd -m "$USERNAME"
        echo "Créé : $USERNAME"
    elif [ "$ACTION" = "delete" ]; then
        # Supprime l'utilisateur et son répertoire home (-r)
        userdel -r "$USERNAME"
        echo "Supprimé : $USERNAME"
    else
        # Si l'argument n'est ni create ni delete
        echo "Veuillez entrer 'create' ou 'delete'"
    fi

    # Incrémente le compteur de 1
    i=$((i + 1))
done
```
